# 🚀 Primeiro Projeto - HTML e CSS

## 📄 Descrição
Este é o primeiro projeto desenvolvido durante o curso **Fullstack PRO**, com foco na construção de uma página web utilizando HTML e CSS.  
O projeto também inclui **responsividade**, garantindo uma boa experiência em diferentes dispositivos.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- Responsividade (Media Queries / Layout adaptável)

## 📚 O que foi aprendido
- Estruturação de páginas com HTML
- Uso de tags semânticas
- Estilização com CSS
- Seletores, classes e IDs
- Box Model (margin, padding, border)
- Organização de layout
- Responsividade para diferentes telas

## ▶️ Como executar o projeto
1. Clone este repositório:
   ```bash
   git clone <URL_DO_REPOSITORIO>
````

2. Acesse a pasta do projeto:

   ```bash
   cd nome-do-projeto
   ```

3. Abra o arquivo `index.html` no navegador:

   * Clique duas vezes no arquivo
     **ou**
   * Use uma extensão como Live Server (VS Code)

## 📁 Estrutura básica

```
/projeto
│── index.html
│── style.css
│── /assets (imagens, ícones, etc)
```
